# Stopwatch Web Application

## SkillCraft Technology Internship - Task 02

### Project Description
A responsive stopwatch web application built using HTML, CSS, and JavaScript. It allows users to accurately measure time intervals with start, pause, reset, and lap functionalities.

### Features
- Start Stopwatch
- Pause Stopwatch
- Reset Stopwatch
- Record Lap Times
- Responsive Design
- Modern User Interface

### Technologies Used
- HTML5
- CSS3
- JavaScript
